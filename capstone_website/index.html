Algorithm Steps

The path from long-form to short-form content

  • Assess each secondâ€™s likelihood to be important
  • Create a running average of subsequent importance
  • Find top events by probability
  • Use scene detection & object counting to determine clip boundaries
  • Determine # clips used in final video (running sum of length)
  • Encode video by assembling clips in chronological order






Classification

Calculating the probability of each second being a meaningful event

Fast Fourier Transform

Frequency and amplitude measured for each second of footage for over 30 complete soccer games.

Principle Component Analysis

FFT data transformed into 170-dimensions via PCA to allow for classification and to avoid overfitting.

Ensemble Method

Weighted model anchored by Gradient Boosting, K-Nearest Neighbors, Random Forest and SVM classifiers.

Over 93% accuracy when trying to predict goals vs. non-goals.
90% accuracy when including missed goals.







Finding Boundaries

Picking the best start/stop points for a clip

  • Start from peak
  • Find the lowest points on both sides
  • Find the first scene that starts at least 30 seconds before the peak.
  • Pick the earliest of the 2 & 3
  • Find the first scene after the peak with over 100 objects (using SURF)
  • Pick the latest of 2 & 5 (scene end)

Next Steps

Iteratively improving the product, building out the pipeline

  • Iterate: Get training data to improve precision and recall for complete videos using Levenshtein (edit-distance)
  • Improve clip boundary determination by rating each scene and optimizing the complete summary clip
  • Fully automate summary generation





Delivering Sports Summaries to Consumers

Consumers in control

  • PVR (personal video recorder) -- run locally
  • Content provider -- run on a server
  • Store index in the cloud -- index factory + highlight app
  • Sports channel -- Facebook-like curated content



Generalizability to other sports

  • A crowd is a crowd: they speak the same language
  • Different games have different narratives:
    • fooball: drive leading up to score
    • basketball: short, stand-alone clips
    • golf: include silence prior to the highlight
  • Some sports to not have a crowd, or a reaction is not audible (e.g., some olympic sports: downhill skiing, swimming)
  • Conclusion: with adjustments, application can generalize for all popular spectator sports
